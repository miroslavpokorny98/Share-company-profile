This is a research and planning directory for **VKZ** (a web design/development studio) working on a website redesign project for **Share** (share.inc) — a Kenyan B2B internet infrastructure company. The project ID is PROP-217. There is no code here — only structured research documents used to inform design, content strategy, and the client proposal.

## Key Context

**Client:** Share (share.inc) — Africa's bandwidth marketplace / hyperscale internet network. Pre-Series A. Primary audience is ISP partners (B2B), secondary is investors.

**Agency:** VKZ (vkz.studio) — the party doing design and development.

**Proposed tech stack:** React/Next.js + Sanity CMS + Three.js/WebGL + Vercel hosting.

**Current site:** 3 pages on Webflow. Being fully replaced.

**Key brand notes:** Typefaces are Aeonik Pro + HK Requisite. Dark cinematic aesthetic, deep blues and blacks, "ribbed glass" texture. Open to subtle refresh emphasizing physical infrastructure.

**Numbers to verify before publishing:** 7 subsea cables, 400+ km fiber, 8M+ people covered, 7 micro data centers — all flagged as needing confirmation from Share. _Web research (March 18) found no independent sources for these numbers. Kenya has 8 cables total, so Share's "7 interconnections" needs clarification. See `research/06_web_research_verification.md` for full audit._
